tomcat转html的使用方案

1.docker pull tomcat (默认是下载最新的)

2.创建文件，替换旧的文件 (vim webapps/ROOT/index.html)

3.启动容器

4.docker run -v /root/duan/webapps/ROOT/:/usr/local/tomcat/webapps/ROOT/ --name tom-duan -itd -p 8181:8080 tomcat

使用到的工具是：sublime
