### 目录创建
#### 方法1（在专业软件中使用）
```
    1.在你写的目录上面写[toc]
    2.使用#号来写你的目录
    3.这样目录就会自动创建了（IDEA中不支持）
```
#### 方法2（在gitHub上展示最好）
```
      1.手写目录
      2.格式：[word](#标题名)
```
### 图片插入
```
  1.格式 ![](image path)
  2.图片最好放在一个文件夹下，这样以后好调用
```
### 网页连接写法
```
   1.格式：[文字描述](网址)
   2.格式 [word](url)
```
